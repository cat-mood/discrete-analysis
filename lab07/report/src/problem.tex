\CWHeader{Лабораторная работа \textnumero 7}

\CWProblem{
    Задана матрица натуральных чисел $A$ размерности $n \times m$. Из текущей клетки можно перейти в любую из 3-х соседних, стоящих в строке с 
    номером на единицу больше, при этом за каждый проход через клетку $(i, j)$ взымается штраф $A_{i,j}$. 
    Необходимо пройти из какой-нибудь клетки верхней строки до любой клетки нижней, набрав при проходе по клеткам минимальный штраф.
}
